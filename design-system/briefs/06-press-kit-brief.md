# Brief 06 · PixCull v1.0 Press Kit

> **Hand to a marketing studio** (small-team agency comfortable
> with launches of Linear / Notion / Stripe scale).  4-6 weeks,
> RMB 10,000–20,000.  Phase C.  Trigger: v1.0 release window.

## Why a press kit

PixCull's v1.0 release coincides with the public broadcast moment
(launch post · Hacker News · 小红书 · 36kr coverage · 摄影师社群
groups · ModelScope feature placement).  Without a press kit:

- Bloggers / reviewers write off-brand because they don't have
  on-brand assets at hand
- Press releases use unflattering screenshots because that's all
  they can find
- Translations of the product narrative drift from the source

The press kit fixes all three by giving every downstream
amplifier **a single zip file** with high-resolution everything.

## Scope — the kit contents

### A. Visuals (15-20 hi-res files)

- **Brand mark** — 5 sizes (16 / 32 / 256 / 1024 / 4096 px PNG +
  SVG sources)
- **Logomark + wordmark** lockup — horizontal / vertical / stacked
  / mono-on-dark / mono-on-light variants
- **Hero screenshots** — 5 frames of PixCull rendering real
  data, each at 1440×900 retina (4× = 2880×1800)
  - `/results` grid view with mixed photo types
  - `/results` lightbox with inspector pane
  - `/share/<token>` portfolio cover
  - `cli_audit --executive` PDF cover render
  - iPad lightbox with swipe gesture mid-flight
- **Product photo composites** — 3 frames showing PixCull "in
  context": laptop on a photographer's desk · iPad held in
  ceremony venue · MacBook + DSLR tether cable
- **Brand pattern** — repeating motif (the 5-circle "spotlight"
  mark scaled up to background-tile size) for marketing graphics

### B. Video assets (3 clips)

- **Trailer · 30s** — Rive-driven hero reveal + Cmd+K demo + LAN
  sync demo + executive PDF generation.  Music optional.
- **Walk-through · 90s** — narrated walkthrough of the v1.0
  feature set, voiceover-friendly
- **Loop · 5s GIF** — single feature highlight (hero reveal),
  social-media friendly, < 500 KB

Each delivered as:
- MP4 1080p H.264 + AAC (universal)
- MP4 4K H.265 (high quality)
- WebM VP9 (web embed)
- (Trailer only) GIF compressed via gifsicle

### C. Written copy (5 documents)

1. **One-pager PDF** (single A4) — the elevator pitch.  Sub-pitch:
   "Cull your wedding in coffee-time, not all-night-time"
2. **Long-form blog post (1800-2400 words)** — the "Why we built
   PixCull" piece.  Author voice: photographer-turned-engineer.
   Tone reference: Linear's "Slow software" + Notion's "AI
   blocks" launch posts.
3. **Three short-form variants** — for HN front page (snappy,
   technical), 小红书 (photographer-voice, image-driven), 36kr /
   tech press (product-market-fit framing)
4. **FAQ doc** — anticipated press questions + on-message answers
5. **Press contact + author bio** — hello@pixcull.dev +
   GitHub + ModelScope + short 50-word bio

### D. Boilerplate

- **Project description** — 30-word / 100-word / 300-word
  versions, English + Simplified Chinese
- **Tagline** — 5-7 word brand promise (current: "本地优先 AI 选片"
  — refresh in collab with copywriter)
- **License + open-source statement** — pre-formatted for
  inclusion in any third-party article

## Style + voice constraints

- **Photographer-voice anchoring** — every line should sound
  like a peer working photographer talking about a tool they use,
  not a software product introducing itself.  Anti-patterns to
  REJECT: "machine-learning-powered", "next-generation",
  "intelligent", "smart" — these read as AI-marketing template.
  Better: "1,500 shots, 30 minutes, you're done"
- **Localised separately, not auto-translated** — Chinese version
  isn't a translation of the English; both have a native author.
- **No marketing-em-dashes or rhetorical flourishes** — read each
  sentence aloud; if it sounds like a script, rewrite.

## Acceptance criteria

- Every file is **brand-guide-compliant** (see Brief 01)
- The trailer (30s clip) generates ≥ 1.0M views across launch
  channels in the first month (aspirational; tracked, not
  contractual)
- The one-pager PDF prints cleanly + reads on phone screen
- Boilerplate copy passes the "blind 5-second test" — a stranger
  who hasn't heard of PixCull understands within 5 seconds what
  it does + who it's for
