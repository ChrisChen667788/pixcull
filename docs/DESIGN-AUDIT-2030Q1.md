## 一、实测发现的真 bug(本轮起点)

v2.10 抓图时实测:近重复折叠 / 🎬 时序场景 的切换 pill 被注入 `#burstPeakFilter`,
而它在 `<details id="lpBurstGroup" style="display:none">`(左侧栏「连拍」组,**默认
隐藏**)里——无连拍的 run(如 relicdemo)整组消失,这些通用透明度功能**零入口**。
连带发现:相似度滑块 CSS 作用域写成 `.filters`,但 pill 实际在侧栏 `.lp-content`,
**自 v2.9 起滑块一直没样式**。两个 bug 都在本轮修掉。
